# Homelab Roadmap

## Current State
- Ubuntu Server host
- Docker + Docker Compose
- Media automation stack (Jellyfin, Radarr, Sonarr, SABnzbd)
- Persistent storage using bind mounts
- Secure remote access
- System monitering

## Short-Term Goals
- Add documentation for each service
- Add a storage layout diagram
- Add a troubleshooting playbook

## Mid-Term Goals
- Prometheus + Grafana monitoring
- Centralized logging
- Automated backups and snapshots
- Reverse proxy (Traefik or Nginx Proxy Manager)

## Long-Term Goals
- Kubernetes (K3s or MicroK8s)
- Local AI inference services
- Multi-node clustering
- Infrastructure-as-code (Ansible or Terraform)
